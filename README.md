# QuizNova - Learn & Play

## Project Introduction
**QuizNova** is a modern, interactive web-based quiz platform designed for students to master fundamental academic subjects. The project demonstrates a clean UI/UX with real-time feedback, progress tracking, and session history. It is built using pure HTML, CSS, and Vanilla JavaScript to ensure high performance and educational simplicity.

## Project Pages
1. **Home (`index.html`)**: The entry point where users can select from various subjects like Programming (C++), Discrete Math, ICT, and more.
2. **About / Instructions (`about.html`)**: Provides background on the project, introduces the team, and explains the game rules and scoring system.
3. **Quiz Mode (`quiz.html`)**: The core interactive experience featuring 10 randomized questions per topic, a 15-second countdown timer, and a visual progress bar.
4. **Results (`result.html`)**: displays the final score and awards a performance badge based on the user's accuracy.
5. **Quiz History (`history.html`)**: Stores and displays the user's recent quiz attempts in the current session using local storage.
6. **Feedback / Help (`feedback.html`)**: A dedicated page for users to submit feedback or request assistance.

## Technology Stack
- **Structure**: HTML5 (Semantic tags)
- **Styling**: Vanilla CSS3 (Custom properties, Flexbox/Grid, Glassmorphism)
- **Logic**: Vanilla JavaScript (ES6+ features, LocalStorage API)
